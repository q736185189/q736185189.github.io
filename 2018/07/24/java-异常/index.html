java(异常)

发表于 2018-07-24 更新于 2019-07-24

1)异常概念
异常时程序在运行过程中出现的不正常现象
java将常见的不正常现象抽象为异常类(Exception)
异常类Exception继承了Throwable类因此所有的异常都是可以抛出的
2）异常分类
异常分为运行时异常，编译时异常两种
1.编译时异常
其实所有异常都是在运行时出现的，这里说的编译时异常时需要在程序编写时预处理的
编译时异常的预处理
1) throws抛出处理
2) try…catch捕获处理
一般情况下,在调用方法时,需要对受检异常进行预处理就选择捕获处理.
如果确定这段代码在运行时不会产生异常,也可以选择抛出处理
2.运行时异常
运行时异常可以通过规范的代码编写避免
